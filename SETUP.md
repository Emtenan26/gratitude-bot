# 🌸 دليل تشغيل بوت الامتنان — خطوة بخطوة

---

## 📁 ملفات المشروع

```
gratitude_bot/
├── bot.py           ← الكود الرئيسي للبوت
├── database.py      ← قاعدة البيانات
├── requirements.txt ← المكتبات المطلوبة
├── render.yaml      ← إعدادات الاستضافة
└── SETUP.md         ← هذا الدليل
```

---

## الخطوة 1 — احصل على Anthropic API Key (مجاني)

1. افتح [console.anthropic.com](https://console.anthropic.com)
2. سجّل حساب جديد (أو سجّل دخول)
3. اضغط **API Keys** من القائمة الجانبية
4. اضغط **Create Key** واحفظ المفتاح — شكله: `sk-ant-...`

> ⚠️ هتحتاج تضيف بطاقة بنكية لتفعيل الـ API — التكلفة رخيصة جداً (~5 دولار كل آلاف الرسائل)

---

## الخطوة 2 — ارفع الكود على GitHub

1. افتح [github.com](https://github.com) وسجّل حساب لو مش عندك
2. اضغط **New Repository** → اسمه `gratitude-bot`
3. حمّل الملفات الأربعة: `bot.py`, `database.py`, `requirements.txt`, `render.yaml`

---

## الخطوة 3 — استضف البوت على Render (مجاني)

1. افتح [render.com](https://render.com) وسجّل دخول بحساب GitHub
2. اضغط **New** → **Blueprint**
3. اختر الـ repo اللي رفعته
4. Render هيقرأ ملف `render.yaml` تلقائياً
5. في خانة **Environment Variables** أضف:
   - `TELEGRAM_TOKEN` = التوكن بتاعك من BotFather
   - `ANTHROPIC_API_KEY` = المفتاح من الخطوة 1
6. اضغط **Deploy** وانتظر دقيقتين

---

## الخطوة 4 — اختبر البوت

1. افتح تيليجرام وروح للبوت بتاعك
2. اكتب `/start`
3. المفروض تجيلك رسالة ترحيب ✅

## ❓ مشاكل شائعة

**البوت مش بيرد؟**
- تأكد من صحة `TELEGRAM_TOKEN`
- اتأكد إن الـ service على Render شغّال (الضوء الأخضر)

**خطأ في Anthropic API؟**
- تأكد إن عندك رصيد في الحساب
- تأكد من صحة `ANTHROPIC_API_KEY`

**الرسائل مش بتوصل بالوقت الصح؟**
- Render بيستخدم UTC تلقائياً — الكود عنده ضبط توقيت مصر تلقائي ✅
